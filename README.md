# Capstone-project-Campervan-users
Figma wireframe showing the DriveAway web application user interface for a campervan driveway booking service. The main section features a hero area with a large green banner labeled Hero Section and a caption describing the service. Below are four listing cards each labeled Listing Title with checkboxes and price badges. A floating Map View button is present. The top right corner has Login/Register buttons, which change to a user icon dropdown with Manage Listings, Manage Bookings, and Logout options after login. Arrows and annotations explain user flows for managing listings and bookings, including uploading listings, blocking availability, confirming or cancelling bookings, and editing details. The wireframe also shows navigation to Manage my Listings and Manage my Bookings pages, each with CRUD functionality for listings and bookings. A calendar interface is included for managing availability, and booking statuses are shown as unconfirmed, confirmed, or cancelled. The emotional tone is neutral and instructional, focusing on clarity and usability. All interface text and annotations are transcribed in the description.
